# Firma_Digital_RSA

Repositorio de ejemplo que contiene scripts para firmar y verificar documentos usando RSA y la librería `cryptography`.

Uso rápido:

- Crear y activar entorno virtual (Windows PowerShell):
  ```powershell
  py -3 -m venv venv
  .\venv\Scripts\Activate.ps1
  ```
- Instalar dependencias:
  ```powershell
  pip install -r requirements.txt
  ```
- Ejecutar ejemplos:
  ```powershell
  py firmar_documento.py
  py verificar_documento.py
  ```
